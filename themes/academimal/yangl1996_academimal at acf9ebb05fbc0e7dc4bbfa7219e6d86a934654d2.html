README.md

Academimal

Academimal is a two-column Hugo theme for researchers. It is based on the awesome Jekyll theme Minimal created by orderdlist.

Features

  • Bio
  • Teaching
  • Email
  • PGP pubkey
  • External links

Note that this theme is a WIP. I plan to add the following features as the need comes:

  • Publication
  • Projects
  • i18n

Demo / Example

I use this theme for my own homepage. The source code can be found on GitHub.
